## Linux 环境配置

### Shell 配置

#### ~/.bashrc

```bash
# 别名
alias ll='ls -la'
alias gs='git status'
alias gp='git push'

# PATH
export PATH=$HOME/bin:$PATH

# 提示符
export PS1='\u@\h:\w\$ '
```

#### ~/.bash_profile

```bash
# 加载.bashrc
if [ -f ~/.bashrc ]; then
    source ~/.bashrc
fi

# 环境变量
export EDITOR=vim
export VISUAL=vim
```
