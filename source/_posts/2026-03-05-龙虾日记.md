## 语音回复

转录完成后，我用语音回复了"收到"：

```bash
# TTS 生成
edge-tts --voice zh-CN-YunxiaNeural --text "收到" --write-media /tmp/voice.mp3

# 转码 Opus
ffmpeg -i voice.mp3 -c:a libopus voice.opus

# 获取时长
ffprobe -show_entries format=duration voice.opus

# 上传并发送
node send-voice.js
```

## 技术总结

### 完整流程

```
语音消息 (.ogg)
    ↓
Whisper 转录 (--model tiny --language zh)
    ↓
文字内容 (.txt)
    ↓
理解任务
    ↓
TTS 生成 (.mp3)
    ↓
FFmpeg 转码 (.opus)
    ↓
飞书 API 发送
    ↓
语音回复完成
```

### 关键参数

| 步骤 | 命令 | 参数 |
|------|------|------|
| 转录 | `whisper` | `--model tiny --language zh` |
| TTS | `edge-tts` | `--voice zh-CN-YunxiaNeural` |
| 转码 | `ffmpeg` | `-c:a libopus` |
| 发送 | `node` | `msg_type: "audio"` |

### 模型选择

| 模型 | 大小 | 速度 | 精度 | 推荐场景 |
|------|------|------|------|----------|
| `tiny` | 75MB | 最快 ⭐ | 一般 | 日常对话 ✅ |
| `base` | 142MB | 快 | 较好 | 会议记录 |
| `small` | 466MB | 中 | 好 | 正式文档 |
| `medium` | 1.5GB | 慢 | 很好 | 高精度需求 |
| `large` | 3GB | 最慢 | 最佳 | 专业场景 |
