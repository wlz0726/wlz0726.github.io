🔧 技术实践

1. PDF 内容提取

挑战: 中文 PDF 文件读取

解决方案: `python import PyPDF2

with open(pdf_path, 'rb') as f: reader = PyPDF2.PdfReader(f) for page in reader.pages: text = page.extract_text() print(text) `

关键点:

  • 使用 rb 二进制模式打开
  • 处理中文编码问题
  • 批量提取多页内容

    • ---

    2. PPT 自动化生成

    核心代码: `python from pptx import Presentation from pptx.util import Inches, Pt from pptx.dml.color import RGBColor from pptx.enum.shapes import MSO_SHAPE

    prs = Presentation() prs.slide_width = Inches(13.333) # 16:9 prs.slide_height = Inches(7.5)

    添加幻灯片

    slide = prs.slides.add_slide(prs.slide_layouts[6])

    添加形状

    card = slide.shapes.add_shape( MSO_SHAPE.ROUNDED_RECTANGLE, left, top, width, height ) card.fill.solid() card.fill.fore_color.rgb = RGBColor(255, 255, 255) card.line.color.rgb = RGBColor(41, 98, 255)     `

    设计实现:

  • 16:9 宽屏比例
  • 卡片式布局（圆角矩形）
  • 主题配色（深蓝 + 青绿）
  • 渐变背景模拟
  • 统一字体层级

    3. 视觉设计技巧

    #### 配色方案 `python

    主题色

    PRIMARY = (41, 98, 255) # 科技蓝 SECONDARY = (0, 184, 148) # 青绿色 LIGHT = (230, 240, 255) # 浅蓝背景 DARK = (45, 52, 54) # 深灰文字 `

    #### 渐变背景模拟 用多层半透明矩形叠加模拟渐变效果： `python for i in range(5): rect = slide.shapes.add_shape(...) alpha = 1.0 - i * 0.15 # 逐层降低透明度 `

    #### 阴影效果 用偏移的半透明黑色矩形模拟阴影： `python shadow = slide.shapes.add_shape( MSO_SHAPE.ROUNDED_RECTANGLE, left+0.15, top+0.15, width, height # 偏移位置 ) shadow.fill.transparency = 0.85 ``

    🛠️ 技术笔记

    PyPDF2 vs pdfplumber

    | 特性 | PyPDF2 | pdfplumber | |------|--------|------------| | 安装难度 | ✅ 简单 | ⚠️ 依赖多 | | 中文支持 | ✅ 良好 | ✅ 良好 | | 表格提取 | ❌ 弱 | ✅ 强 | | 速度 | ✅ 快 | ⚠️ 中等 |

    结论: 简单文本提取用 PyPDF2，复杂表格用 pdfplumber

    python-pptx 限制

  • ❌ 不支持真正的渐变填充
  • ❌ 不支持阴影效果
  • ✅ 可用多层矩形模拟
  • ✅ 支持形状、文字、图片
