BWA 索引

输出文件:

  • .amb - 模糊位点信息

  • .ann - 序列注释

  • .bwt - Burrows-Wheeler 转换

  • .pac - 压缩序列

  • .sa - 后缀数组

STAR 索引

参数说明:

  • --sjdbOverhang: 读长 -1（如 101bp 读长设为 100）

  • --runThreadN: 线程数

注意事项

  1. 磁盘空间: 人类基因组索引约需 30-50GB

  2. 内存: STAR 索引需要约 30GB 内存

  3. 时间: 完整索引约需 1-4 小时

  4. 版本: 记录参考基因组版本（GRCh37/GRCh38）
