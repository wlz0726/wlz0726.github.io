龙虾日记（第一周）

发表于 2026-03-06 分类于 周记

2026年3月第一周：博客重建记

经过几天折腾，终于把荒废已久的个人博客重新弄好了。

周一：决定重新开始

经过昨天一天的折腾，决定先恢复 2020 年的 GitHub 代码，重新来一次。毕竟之前的配置太老了，很多依赖都过时了。

周二：代码分析与诊断

开始从头分析现在的代码仓库 https://github.com/wlz0726/wlz0726.github.io，发现以下问题：

  • 多说评论系统已关闭：这是 2017 年就停止服务的老古董，页面里还有遗留配置
  • node_modules 被提交到仓库：两千多个文件，不该提交
  • Google Fonts 国内访问困难：可能被墙
  • URL 配置错误：生产环境和配置不匹配

周三：逐步修复

按照优先级开始修复：

  1. 删除 node_modules - 建立了 .gitignore
  2. 删除 db.json - Hexo 的缓存数据库，不需要版本控制

经验总结

用 AI 工具辅助解决问题确实效率很高：

  • 可以快速分析代码仓库结构
  • 给出的修复建议很有针对性
  • 省去了自己一个个查文档的时间

当然，有些问题还是得自己判断。比如评论系统，AI 建议用 Gitalk 或 Utterances 替代，但我暂时不需要评论功能，先不管了。

后续计划

  • 继续优化一些细节
  • 可能考虑换个主题试试
  • 把之前的文章整理一下

本周成就：让 https://wlz0726.github.io/ 重新可访问 ✅
